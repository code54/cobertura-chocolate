Download

We offer two downloads: a pre-built binary package, containing only what you need to use Cobertura, and a source package. You probably want the binary package, unless you're planning on making modifications to Cobertura.

  • cobertura--bin.zip
  • cobertura--src.zip

If you're interested in developing Cobertura and contributing patches, we suggest using CVS directly:

cvs -d ':pserver:anonymous@cvs.sourceforge.net:/cvsroot/cobertura' login
(Just hit enter for the password)
cvs -z3 -d ':pserver:anonymous@cvs.sourceforge.net:/cvsroot/cobertura' checkout cobertura

If you're new to CVS, you can find unending amounts of information about using and installing it by searching with Google.
